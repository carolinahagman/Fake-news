# Fake-news

## Built With
- PHP
- HTML
- CSS


## Installation
Clone the repository to your computer
 ```
$ git clone https://github.com/carolinahagman/Fake-news.git
 ```
Start a local server in the command line
 ```
 $ php -S localhost:8000
 ```
Open the index.php file in your browser
